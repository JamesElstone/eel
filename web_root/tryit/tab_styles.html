Tab Style Tryout

Pure HTML and CSS visual options for EEL Accounts. The selected states are static here, so the page is useful for judging shape, density, colour, and how the tabs sit above accounting content.

1. Classic Rounded Tabs

Best default

Clear card-level navigation

This is closest to the current app language: restrained, obvious, and easy to scan inside a dense accounting workflow.

3 open checks Ledger ready

Recommended for most pages where users switch between related card views.

2. Curved Tapered Tabs

Most striking

Curved with angled shoulders

The pseudo-elements create tapered edges while border-radius keeps the shape soft enough for the existing interface.

FY 2025 Needs review

Good when you want a distinctive page section without making the whole app feel decorative.

3. Folder Tabs

Auditable feel

Document cabinet metaphor

This shape works well for upload flows, filing packs, statements, and drill-downs where the user expects documents or records.

2 files 1 issue

A strong candidate for uploads and year-end packs, but probably too literal for every page.

4. Ledger Pills

Compact

Best for secondary switches

Segmented-control tabs are efficient for small mode changes, filters, and compact card headers.

128 rows Balanced

Use inside a card or toolbar; avoid for primary page navigation because it can look like a filter.

5. Status Signal Tabs

Workflow-heavy

Tabs with operational status

This pattern carries counts and risk state directly in the tab, useful for transaction import and reconciliation screens.

Ready to post VAT review

Powerful for busy pages, but reserve it for places where the status genuinely changes decisions.

6. Ribbon Taper Tabs

Sharpest silhouette

Angular and prominent

This has the most visual bite. It is memorable, but it should be used carefully in a professional accounting surface.

Step 1 of 3 Validation pending

Best as a one-off tryout or wizard header; less suitable as the app-wide tab pattern.

Recommendation for EEL Accounts

The best app-wide choice is a restrained rounded tab with a slightly tapered variant available for high-value workflows. It stays professional, fits the current card system, and does not compete with status colours that already carry accounting meaning.

Primary tabs

Use Classic Rounded Tabs for page sections such as Overview, Transactions, Reconciliation, and Audit.

Striking variant

Use Curved Tapered Tabs where you want a stronger page identity, especially period or year-end workspaces.

Status workflows

Use Status Signal Tabs only when the tab label needs counts, warnings, or readiness state.
